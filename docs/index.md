# UltraLiteDB

UltraLiteDB is an ultra-lightweight embedded .NET NoSQL document store in a single data file — a fork of [LiteDB](https://www.litedb.org/).
It is built to use in Unity, but could be useful in any dotnet project. It also supplies a mature [BSON](https://bsonspec.org) serializtion system with support
for attributes, custom serializers, as well as JSON conversion.
